# Day 89 - Coding Challenge

## Question 1  
**Problem:**  
Show that enums store integers by printing assigned values.

**Code:**  
```c
#include <stdio.h>

enum Status
{
    SUCCESS, // 0
    FAILURE, // 1
    TIMEOUT  // 2
};

int main()
{
    printf("SUCCESS=%d, FAILURE=%d, TIMEOUT=%d", SUCCESS, FAILURE, TIMEOUT);
    return 0;
}

```

**Sample Output:**  
```
SUCCESS=0, FAILURE=1, TIMEOUT=2
```
