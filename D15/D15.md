# Day 15 - Coding Challenge

## Question 1  
**Problem:**  
Write a program to calculate the factorial of a number.

**Code:**  
```c
#include <stdio.h>
int main(){
    int n, factorial = 1;
    printf("Enter n: ");
    scanf("%d", &n);
    for(int i = 1; i <= n; i++) {
        factorial *= i;
    }
    printf("Factorial of %d is: %d\n", n, factorial);
    return 0;
}
```

**Sample Output:**  
```
Enter n: 5
Factorial of 5 is: 120
```

## Question 2  
**Problem:**  
Write a program to reverse a given number.

**Code:**  
```c
#include <stdio.h>
int main(){
    int n, reversed = 0;
    printf("Enter n: ");
    scanf("%d", &n);
    while(n != 0) {
        reversed = reversed * 10 + n % 10;
        n /= 10;
    }
    printf("Reversed number is: %d\n", reversed);
    return 0;
}
```

**Sample Output:**  
```
Enter n: 45
Reversed number is: 54
```
