# Day 91 - Coding Challenge

## Question 1  
**Problem:**  
Define a structure Student with name, roll_no, and marks, then read and print one student's data.

**Code:**  
```c
#include <stdio.h>

typedef struct 
{
    char name[50];
    int roll;
    int marks;
} Student;

int main(){
    Student s;
    printf("Enter name: ");
    scanf("%s",s.name);

    printf("Enter Roll No. : ");
    scanf("%d",&s.roll);

    printf("Enter Marks : ");
    scanf("%d",&s.marks);


    printf("%s|%d|%d",s.name,s.roll,s.marks);
    return 0;
    
}
```

**Sample Output:**  
```
Enter name: Asha
Enter Roll No. : 123456
Enter Marks : 100
Asha|123456|100
```
