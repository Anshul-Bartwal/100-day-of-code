# Day 99 - Coding Challenge

## Question 1  
**Problem:**  
Use malloc() to allocate structure memory dynamically and print details.

**Code:**  
```c
#include <stdio.h>
#include <stdlib.h>
typedef struct 
{
    char name[50];
    int roll;
    int marks;
} Student;



    


int main(){
    Student *s = (Student *)malloc(sizeof(Student));

    printf("Enter name: ");
    scanf("%s",s->name);

    printf("Enter Roll No. : ");
    scanf("%d",&s->roll);

    printf("Enter Marks : ");
    scanf("%d",&s->marks);

    printf("  %s  |  %d  |  %d  ",s->name,s->roll,s->marks);
    return 0;


}
```

**Sample Output:**  
```
Enter name: asha
Enter Roll No. : 100
Enter Marks : 89  
  asha  |  100  |  89 
```
