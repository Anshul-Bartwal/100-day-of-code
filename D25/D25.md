# Day 25 - Coding Challenge

## Question 1  
**Problem:**  
Write a program to print the following pattern:
5
45
345
2345
12345

**Code:**  
```c
#include <stdio.h>
#include <math.h>
int main(){
    int n;
    printf("Enter Number of rows: ");
    scanf("%d",&n);
    for (int i=n;i>0;i--){
        for(int j=i;j<=n;j++){
            printf("%d",j);
        }
        printf("\n");
    }
    return 0;
}
```

**Sample Output:**  
```
Enter Number of rows: 5
5
45
345
2345
12345
```

## Question 2  
**Problem:**
Write a program to print the following pattern:
```
*****
 ****
  ***
   **
    *
```
**Code:**  
```c
#include <stdio.h>
#include <math.h>
int main(){
    int n;
    printf("Enter Number of rows: ");
    scanf("%d",&n);
    for (int i=0;i<n+1;i++){
        //spaces
        for(int s=0;s<i;s++){
            printf(" ");
        }
        //stars
        for(int j=1;j<=n-i;j++){
            printf("*");
        }
        printf("\n");
    }
    return 0;
}
```

**Sample Output:**  
```
Enter Number of rows: 5
*****
 ****
  ***
   **
    *
```
