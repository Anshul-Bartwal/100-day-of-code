# Day 94 - Coding Challenge

## Question 1  
**Problem:**  
Write a function that accepts a structure as parameter and prints its members.

**Code:**  
```c
// Your C solution here#include <stdio.h>
typedef struct 
{
    char name[50];
    int roll;
    int marks;
} Student;


void display(Student s){
    printf("  %s  |  %d  |  %d  ",s.name,s.roll,s.marks);

}
int main(){
    Student s;

    printf("Enter name: ");
    scanf("%s",s.name);

    printf("Enter Roll No. : ");
    scanf("%d",&s.roll);

    printf("Enter Marks : ");
    scanf("%d",&s.marks);


    display(s);
    return 0;


}
```

**Sample Output:**  
```
Enter name: Ansh
Enter Roll No. : 123456
Enter Marks : 100
  Ansh  |  123456  |  100 
```
