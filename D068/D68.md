# Day 68 - Coding Challenge

## Question 1  
**Problem:**  
Write a program to take an input array of size n. The array should contain all the integers between 0 to n except for one. Print that missing number

**Concept Used**
- Gaussian Sum (sum of first n natural numbers)

**Code:**  
```c
#include <stdio.h>

int main()
{
    int len;

    printf("Enter the length of array: \n");
    scanf("%d", &len);
    int nums[len];
    int expectedSum = (len * (len + 1)) / 2;
    int actualSum = 0;
    for (int i = 0; i < len; i++){
        printf("Enter element %d: \n", i + 1);
        scanf("%d", &nums[i]);
        actualSum += nums[i];
    }
    printf("Missing: %d", expectedSum -actualSum);
    return 0;
}
```

**Sample Output:**  
```
Enter the length of array: 
4
Enter element 1: 
0
Enter element 2: 
3
Enter element 3: 
2
Enter element 4: 
4
Missing: 1
```
