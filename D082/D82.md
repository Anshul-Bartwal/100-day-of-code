# Day 82 - Coding Challenge

## Question 1  
**Problem:**  
Define an enum for traffic lights (RED, YELLOW, GREEN) and print 'Stop', 'Wait', or 'Go' based on its value.


**Code:**  
```c
#include <stdio.h>

typedef enum{
    RED,
    YELLOW,
    GREEN
} Light;
int main(){
    Light l=RED; // change to YELLOW or green  fr different outputs
    switch (l)
    {
    case RED:
        printf("Stop");
        break;
    
    case YELLOW:
        printf("WAIT");
        break;
    case GREEN:
        printf("GO");
        break;
    default:
        break;
    }

    return 0;
    
}
```

**Sample Output:**  
```
Stop
```
