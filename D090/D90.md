# Day 90 - Coding Challenge

## Question 1  
**Problem:**  
Define a struct with enum Gender and print person's gender.

**Code:**  
```c
#include <stdio.h>
enum Gender
{
    MALE,
    FEMALE
};

struct Person
{
    char name[50];
    enum Gender gender;
};
int main()
{
    struct Person p = {"John", MALE};
    if (p.gender == MALE){
        printf("Male");
    }
    else{
        printf("Female");
    }
    return 0;
}

```

**Sample Output:**  
```
Male
```
