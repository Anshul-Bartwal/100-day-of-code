# Day 49 - Coding Challenge

## Question 1  
**Problem:**  
Print the initials of a name.

**Code:**  
```c
#include <stdio.h>
int main(){
    char name[100];
    printf("Enter name: ");
    fgets(name,sizeof(name),stdin);
    int i=0;
    while(name[i] != '\0'){

        if (i==0){
            printf("%c ",name[0]);
        }
        else if (name[i] == ' '){
            printf("%c ",name[i+1]);
        }
        i++;
    }
    return 0;
}
```

**Sample Output:**  
```
Enter name: John Doe
J D
```

## Question 2  
**Problem:**  
Print initials of a name with the surname displayed in full.

**Code:**  
```c
#include <stdio.h>
#include <string.h>

int main() {
    char name[100];
    printf("Enter name: ");
    fgets(name, sizeof(name), stdin);

    name[strcspn(name, "\n")] = '\0';

    int i = 0;
    int lastSpace = -1;

    for (i = 0; name[i] != '\0'; i++) {
        if (name[i] == ' ')
            lastSpace = i;
    }

    for (i = 0; name[i] != '\0'; i++) {
        if (i == 0) {
            printf("%c. ", name[i]);  
        } else if (name[i] == ' ' && i < lastSpace) {
            printf("%c. ", name[i + 1]);  
        }
    }

    if (lastSpace != -1)
        printf("%s\n", &name[lastSpace + 1]);

    return 0;
}

```

**Sample Output:**  
```  
Enter name: John David Doe
J. D. Doe

```
