# Day 26 - Coding Challenge

## Question 1  
**Problem:**  
Write a program to print the following pattern:
```
    5
   45
  345
 2345
12345
```

**Code:**  
```c
#include <stdio.h>
int main(){
    int n;
    printf("Enter number of rows: ");
    scanf("%d",&n);
    for (int i=n;i>0;i--){
        for (int s=1;s<i;s++){
            printf(" ");
        }
        for(int j=i;j<=n;j++){
            printf("%d",j);
        }
        printf("\n");
    }
    return 0;
    
}

```

**Sample Output:**  
```
Enter number of rows: 5
    5
   45
  345
 2345
12345
```

## Question 2  
**Problem:**  
Write a program to print the following pattern:
```
*

*
*
*

*
*
*
*
*

*
*
*

* 
```

**Code:**  
```c
#include <stdio.h>
int main(){
    int n;
    printf("Enter max length: ");
    scanf("%d",&n);
    //upper half
    for (int i=1;i<=n;i+=2){
        for(int j=0;j<i;j++){
            printf("*\n");
        }
        printf("\n");


    }
    //lower half
    for(int i=n-2;i>0;i-=2){
        for(int j=0;j<i;j++){
            printf("*\n");
        }
        printf("\n");
        
    }
    return 0;

    
}

```

**Sample Output:**  
```
Enter max length: 5
*

*
*
*

*
*
*
*
*

*
*
*

*
```
