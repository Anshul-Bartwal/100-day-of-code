# Day 85 - Coding Challenge

## Question 1  
**Problem:**  
Assign explicit values starting from 10 and print them.

**Code:**  
```c
#include <stdio.h>

typedef enum {
    A = 10,
    B,
    C
} Letters;

int main() {

    printf("A=%d\n", A);
    printf("B=%d\n", B);
    printf("C=%d\n", C);

    return 0;
}

```

**Sample Output:**  
```
A=10
B=11
C=12
```
