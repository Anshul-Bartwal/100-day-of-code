# Day 88 - Coding Challenge

## Question 1  
**Problem:**  
Print all enum names and integer values using a loop.

**Code:**  
```c
#include <stdio.h>
enum Signal
{
    RED,
    YELLOW,
    GREEN
};
int main()
{
    enum Signal s;
    const char *names[] = {"RED", "YELLOW", "GREEN"};
    for (s = RED; s <= GREEN; s++)
    {
        printf("%s=%d\n", names[s], s);
    }
    return 0;
}

```

**Sample Output:**  
```
RED=0
YELLOW=1
GREEN=2
```
